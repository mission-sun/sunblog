Hexo搭建Github静态博客

2017-03-23
  • 技术类
  • 技术

  1. 环境环境
    1.1 安装Git
    1.2 安装node.js

    上述环境我就不写了，比较简单！

  2. 安装Hexo
    打开Git命令行，执行如下命令

    1. 安装主题
      $ git clone https://github.com/A-limon/pacman.git themes/pacman
      修改你的博客根目录/D/Hexo下的config.yml配置文件中的theme属性，将其设置为pacman
    2. 然后就是把博客搭建到搭建github
常用的hexo执行程序
{% codeblock %}
     	hexo n == hexo new
		hexo g == hexo generate
		hexo s == hexo server
		hexo d == hexo deploy{% endcodeblock %}






第二部分
      介绍如何把自己做的项目放到github上 虽然这跟上边的差不多，但是还是有
      点区别的

    下边我把核心内容记录下来，方便自己查阅
1、创建一个本地项目
    本地项目的文件夹名字任意
2、建立本地仓库
    执行{% codeblock %}git init{% endcodeblock %}
    初始化成功后你会发现项目里多了一个隐藏文件夹.git
3、将自己的文件放到自己建立的文件项目中
    执行{% codeblock %}git add .{% endcodeblock %}
    注意的是上边是个点 ，不要拉掉
 4、然后，把文件提交到仓库，双引号内是提交注释
    执行{% codeblock %} git commit -m "提交文件"{% endcodeblock %}

    本地建立好网站之后，我们需要关联github仓库
    到github上仓库复制我们需要的地址（ssh）
    然后执行{% codeblock %} git remote add origin  仓库地址{% endcodeblock %}
    上传代码
    执行{% codeblock %}git push -u origin master{% endcodeblock %}
    如果上边的程序执行错误的话我们需要执行下边的程序
    {% codeblock %} git push -f origin master {% endcodeblock %}
